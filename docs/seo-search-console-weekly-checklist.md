# Search Console Weekly Checklist

## Every Monday

1. Coverage report
- Check new errors.
- Validate fixed issues.
- Confirm no accidental noindex pages.

2. Indexing status
- Confirm priority URLs are indexed.
- Inspect newly published pages.
- Request indexing for updated pages.

3. Performance report
- Compare last 7 days vs previous 7 days.
- Track clicks, impressions, CTR, average position.
- Export query and page data for clustering.

4. Core Web Vitals / Page Experience
- Review mobile and desktop status.
- Confirm no newly regressed templates.

5. Enhancements and manual actions
- Ensure no manual action flags.
- Verify structured data is parsable.

## Monthly pruning pass

- Merge pages with overlapping intent.
- Redirect the weaker page to the stronger page.
- Update sitemap and internal links.
